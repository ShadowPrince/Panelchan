So, you are finally here. It's pretty straightforward from now on - 1 finger tap shows next page, 2 finger tap shows/hides status bar (there's also second controls layout in the Settings)

The Lock captures the position and zoom, and image will revert to that values when you change the page (opposed to using auto-zoom and top-left image position). This is really usefull when you read manga.

TL/DR: tap - next page, 2 finger tap - hide bar, lock icon locks zoom/position for next page.
